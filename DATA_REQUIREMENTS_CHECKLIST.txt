================================================================================
DATA FORMAT REQUIREMENTS
================================================================================

PREFERRED FORMAT: CSV or Excel

Required columns:
  - date          (YYYY-MM-DD format, e.g., 2024-01-15)
  - symbol        (e.g., "rebar_uae_import", "iron_ore_62fe_cfr_china")
  - price_usd_mt  (numeric value)

Optional columns:
  - price_low, price_high, volume, source

Example:
  date,symbol,price_usd_mt
  2024-01-01,rebar_uae_import,607.50
  2024-01-02,rebar_uae_import,609.25

================================================================================
DATA QUALITY CHECKLIST
================================================================================

Before submitting data, ensure:

  [ ] Completeness: Less than 5% missing values
  [ ] Consistency: Same units throughout (USD/mt)
  [ ] Date format: YYYY-MM-DD (e.g., 2024-01-15)
  [ ] No duplicates: One price per symbol per date
  [ ] Reasonable range: Rebar $300-$1000/mt, Iron ore $60-$180/mt
  [ ] Source documented: Know where each price came from
